# Cats Blender Plugin - Codex 4.0

This repository contains the Codex-maintained Cats Blender Plugin fork prepared for Blender 4.1+.

## Package

- Final installable package: `cats_062.zip`
- Plugin version: `4.0.0`
- Maintainer: `Codex`

## Main changes

- Removed sponsor, credits, and update panels.
- Added selectable Fix Model repair profiles.
- Added model type selection for MMD, VRM, Source, FBX/XPS, and generic models.
- Added IK, FK, and constraint preservation controls.
- Improved armature hierarchy, vertex group, weight, material, UV, and texture repair safety.
- Kept the maintained Chinese localization.

## Validation

The final package was tested in Blender with registration checks, zip install checks, UI localization checks, selectable repair preset checks, and full model repair regression tests.
